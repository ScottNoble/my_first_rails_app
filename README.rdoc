== README
Ruby version 2.0.0
This file is a practice app that we did in class to work on routing and using ruby injections in html. 
There are three main functionalities in this app.
 /time/now you will giv the current time in a custom format
 /hello will load a page that says "Hello World"
 /hello/name will load a page that says "Hello Name"
 /introduce/name1/and/name2 will load a page that says "Hello Name1, have you met Name2? Name1, this is Name2."
